# poem generator using streamlit + LangChain + HuggingFace for closeAI api

User can dyamially choose a poem type , style and length if provided customs or there are few options already

# Features
-Generating poems on different topics
-can control length
-can chose style
-simple and clean
-secure api key with .env

# Tech Stack
- **Python**
- **Streamlit**
- **LangChain**
- **Hugging Face Inference API**
- **Gemma-2-9B-IT model**

# Prompt
i use a dynamic prompt , if you want to change you can change prompt according to your style from prompt_template.py
